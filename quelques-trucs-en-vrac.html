Quelques trucs en vrac

Tropez le jeu. 07 décembre 2006

Kubuntu Edgy Thunderbird Firefox cups amarok permissions mysql

Un disque dur neuf, une kubuntu Edgy Eft fraîchement installée, il reste quelques petits réglages a effectuer pour que tout soit parfait. Voici donc quelques trucs en vrac que je poste ici afin de ne plus avoir à les chercher le cas échéant. Ils paraîtront certainement évidents à beaucoup, on les trouve ailleurs sur internet ou dans la documentation (heureusement pour moi) mais ils ont occupé quelques soirées de l'amateur "éclairé" peut être mais non informaticien que je suis.

  • Utiliser Thunderbird par défaut dans Firefox

Ouvrir Firefox
dans la barre d'adresse entrer about:config
cliquer droit sur la page et choisir nouvelle chaîne de caractères
lui donner comme nom : network.protocol-handler.app.mailto
lui donner comme valeur : mozilla-thunderbird (voire simplement thunderbird pour les distributions les plus récentes).

  • Utiliser Firefox par défaut dans Thunderbird

    • pour une distribution basée sur Debian, taper dans une console avec les droits root
      update-alternatives --config x-www-browser
      et choisir l'option correspondant à Firefox;

    • quelle que soit la distribution,
      ouvrir Thunderbird
      aller dans "Edition" "Préférences"
      Section "Avancé", onglet "général", cliquer sur Editeur de configuration
      cliquer droit et choisir nouvelle chaîne de caractères
      lui donner comme nom : network.protocol-handler.app.http
      lui donner comme valeur : mozilla-firefox (voire simplement firefox pour les distributions les plus récentes).

  • Utiliser Firefox et Thunderbird par défaut dans les applications de KDE

Définir ces deux logiciels respectivement comme navigateur et lecteur de courrier par défaut dans le gestionnaire de configuration de KDE (menu K, réglages du système, applications par défaut, dans ma Kubuntu par exemple).

  • Utiliser cups-pdf avec kubuntu (ubuntu en général) et cups en version 1.2.0 ou supérieure

Pour faire apparaître l'imprimante pdf dans la liste des imprimantes disponibles (choix autre type d'imprimante), taper dans une console :
sudo chmod +s /usr/lib/cups/backend/cups-pdf
si vous rencontrez un problème lors du chargement du pilote (choisir imprimante générique postcript), taper dans une console :
sudo foomatic-cleanupdrivers

  • Lire des MP3 avec Amarok

Installer le paquet libxine-extracodecs

  • Modifier les permissions de tous les sous répertoires d'un dossier sans toucher aux permissions des fichiers

Dans une console, se placer dans le dossier concerné et taper par exemple (droits root nécessaires si on se trouve hors de son répertoire utilisateur personnel évidemment)
find . -type d | xargs chmod 755

  • Sécuriser mysql après installation

Au minimum, ajouter un mot de passe à l'administrateur via phpmyadmin par exemple
plus simplement, taper dans une console (avec les droits root faut-il le répéter ;-))
mysql_secure_installation
et répondre aux quelques questions posées.
