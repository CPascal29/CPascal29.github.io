Libdvdcss2, le retour

Tropez le sam. 23 novembre 2019

Ubuntu dvd libdvdcss2

Pour mémoire, libdvdcss2 est une librairie nécessaire sous Linux pour lire les DVD du commerce. Elle est légale et autorisée en France mais pas dans tous les pays d'où le fait qu'elle n'est pas installée par défaut dans les distributions.

Plusieurs possibilités pour l'installer, notamment via le dépôt VideoLAN mais la plus simple est la suivante :

Installer le paquet libdvd-pkg via la logithèque, synaptic ou en tapant dans un terminal : sudo apt install libdvd-pkg

exécuter dans un terminal : sudo dpkg-reconfigure libdvd-pkg

Pour plus de détails : https://doc.ubuntu-fr.org/lire_un_dvd
