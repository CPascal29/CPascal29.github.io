Migration accomplie

Tropez le ven. 21 août 2009

migration Dotclear Free

Après quelques tentatives inabouties, j'ai finalement effectué la migration de la version 1.2.8 vers la version 2.1.5 de Dotclear.

En fait, le plus compliqué a été de repersonnaliser le blog vu qu'il y a incompatibilité entre les thèmes des deux versions. DotAddict m'a été d'un grand secours et une fois que j'ai eu trouvé quelque chose qui se rapprochait de mes préférences, un peu de bidouillage dans les fichiers de style via l'éditeur de thème a fait le reste.

Pour la suite, mes choix ont été dictés par les restrictions liées à l'hébergement pages personnelles de Free :

  • la récupération des données a été effectuée (très simplement) via le plugin flatexport;

  • l'activation de php5 est nécessaire et se fait via la création d'un fichier texte contenant juste php 1 enregistré à la racine du blog sous l'appellation .htaccess;

  • il faut activer la base postgreSQL proposée par Free. Dotclear 2 ne fonctionne pas avec la version disponible de mySQL. Cela ne pose aucun problème pratique si la base mySQL est déjà activée car Free autorise la cohabitation des deux;

  • le plugin fortune n'étant pas supporté par Dotclear 2, j'ai utilisé BashFR Quote pour les citations.

Et voila, le résultat est là.
