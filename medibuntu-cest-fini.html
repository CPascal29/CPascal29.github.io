Medibuntu, c'est fini

Tropez le dim. 11 mai 2014

Medibuntu dépôt libdvdcss2 Libdvdcss2 codec vlc

Adieu Medibuntu, le dépôt est fermé. La plupart des logiciels et codecs fournis sont aujourd'hui disponibles dans les dépôts officiels. Seule exception, Libdvdcss2, accessible dorénavant dans le dépôt vlc.

Pour faire court, l'adresse de ce dernier est :

  • deb http://download.videolan.org/pub/debian/stable

et pour obtenir le certificat :

  • sudo bash
    wget -O - http://download.videolan.org/pub/debian/videolan-apt.asc | sudo apt-key add -

Remarque :
le sudo bash peut paraître surprenant car le wget qui suit devrait se suffire à lui même mais dans ce cas précis, cela ne marche pas si on ne le fait pas.
