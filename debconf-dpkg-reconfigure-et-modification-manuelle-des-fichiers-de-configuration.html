Debconf, dpkg-reconfigure et modification manuelle des fichiers de configuration

Tropez le jeu. 14 septembre 2006

configuration X debconf

Sous Debian, la mise à jour du fichier de configuration du serveur X peut se faire automatiquement via debconf. De même sa modification est très simple grace à dpkg-reconfigure. Cependant, si pour une raison ou pour une autre il est modifié manuellement tout cela ne marche plus.

On trouve cependant, dans l'en-tête de ce même fichier, la procédure à suivre pour pallier ce problème :

  • XFree86 :
    cp /etc/X11/XF86Config-4 /etc/X11/XF86Config-4.custom
    md5sum etc/X11/XF86Config-4 > /var/lib/xfree86/XF86Config-4.md5sum
    dpkg-reconfigure xserver-xfree86\"

  • Xorg :
    dpkg-reconfigure -phigh xserver-xorg

Tout cela avec les droits root bien sur.

  • Questions :
    Dans le cas d'xfree86, si on utilise dpkg-reconfigure -phigh xserver-xfree86 ça marche aussi ?
    De même avec xorg si on utilise md5sum ?
