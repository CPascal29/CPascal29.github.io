Ma configuration

Tropez le lun. 08 août 2005

easinote pentium configuration Linux Windows

J'ai un portable Packard Bell easynote H5310 (processeur Intel pentium 4 à 3,1 Ghz, mémoire vive 512Mo, DD 60Go, écran lcd 15" diamond view, contrôleur vidéo Nvidia Geforce FX Go 5100 128 Mo, contrôleur audio Realtek ALC655, modem Castlenet MM320, port ethernet, firewire, parallèle, 1 port PCMCIA, 3 ports USB2, ports audio, s-vidéo, pas de lecteur de disquette, pas de port série).

Double boot via lilo sur une Debian "testing" ou le Windows XP home fourni. En général, je passe le plus clair de mon temps sous Linux. J'utilise encore le noyau 2.4.27-2 et j'ai installé les pilotes alsa pour profiter pleinement du micro notamment avec skype.

Les quatre ordinateurs présents chez moi sont reliés en réseau via des adaptateurs CPL et partagent la connexion internet (Free non dégroupé, IP ADSL 10272 kbits/s ATM) grace une Freebox en mode routeur. Aucun problème, tout fonctionne à merveille.
