Démarrage en double boot Linux - Windows

Tropez le mer. 02 octobre 2019

Linux Windows boot problème

Cela fait plusieurs fois que je suis confronté au même problème lors d'une installation de Linux en double boot avec Windows sur un ordinateur 64 bits et BIOS UEFI. Tout se déroule sans problème mais au redémarrage, pas de trace de Grub, on bascule directement sur le bootloader Windows.

En fait, la solution est simple. Il suffit d'aller dans le Bios, au niveau de la séquence de démarrage UEFI, on trouve en général une option qui permet de choisir entre booter sur le bootloader Linux ou celui de Windows.
