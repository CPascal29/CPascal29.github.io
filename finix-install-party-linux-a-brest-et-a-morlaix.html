Finix : Install Party Linux à Brest et à Morlaix

Tropez le lun. 16 mars 2015

Install Party Linux fête libre Finix MDL

Le samedi 21 mars 2015 de 14h à 17h, à l'occasion de la fête du libre, Finix Brest organise, en collaboration avec la maison du libre, une Install Party Linux dans les locaux des Fabriques du Ponant, 40 rue Jules Lesven à Brest.

Le samedi 28 mars 2015 (toute la journée à partir de 10h), Finix Morlaix organise une Install Party en association avec la Cyberbase de Morlaix Communauté dans leurs locaux : Cyberbase Morlaix

Venez nombreux découvrir Linux et si vous le désirez l'installer avec notre aide sur votre ordinateur.
